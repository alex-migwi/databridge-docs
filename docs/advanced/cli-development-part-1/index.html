Understanding the Architecture

What is DataBridge?

DataBridge is a code generator that:

  • Reads your MySQL database structure
  • Generates REST API endpoints (Fastify)
  • Generates Angular services (TypeScript)
  • Eliminates hours of boilerplate code

The Flow

MySQL Database â†’ Prisma Schema â†’ Code Generation â†’ Running API + Angular SDK

Simple analogy: Think of DataBridge like a restaurant:

  • Commands = Menu items (init, generate, serve)
  • Flags/Options = Customizations (extra cheese, no onions)
  • Prompts = Waiter asking questions
  • File generation = Kitchen preparing your order

Setting Up the CLI Framework

Why oclif?

oclif is a framework for building professional CLIs. It provides:

  • Command discovery (finds commands in src/commands/)
  • Flag parsing (handles --flag value)
  • Help generation (automatic documentation)
  • Plugin system (extensibility)

Installation

cd packages/cli
pnpm add @oclif/core @oclif/plugin-help inquirer fs-extra chalk
pnpm add -D @types/fs-extra @types/inquirer oclif tsx

Command 1: Init

Purpose

Initialize a DataBridge project by creating:

  • .databridge.json - Configuration file
  • prisma/schema.prisma - Database schema template
  • .env - Environment variables

Key Concepts

1. oclif Command Structure

import { Command, Flags } from '@oclif/core';

export default class Init extends Command {
  static description = 'Initialize a new DataBridge project';
  
  static flags = {
    'skip-prompts': Flags.boolean({
      description: 'Skip interactive prompts',
      default: false,
    }),
  };

  async run(): Promise<void> {
    const { flags } = await this.parse(Init);
    // Command logic here
  }
}

Key parts:

  • static description - Shows in help text
  • static flags - Optional parameters users can pass
  • async run() - Main command logic

2. Interactive Prompts (inquirer)

import inquirer from 'inquirer';

const answers = await inquirer.prompt([
  {
    type: 'input',
    name: 'databaseUrl',
    message: 'MySQL database URL:',
    default: 'mysql://root:password@localhost:3306/mydb',
    validate: (input: string) => {
      if (!input.startsWith('mysql://')) {
        return 'Database URL must start with mysql://';
      }
      return true;
    },
  },
]);

How inquirer works:

  • Shows questions one by one
  • Validates user input
  • Returns answers as an object: { databaseUrl: '...' }

3. File Operations (fs-extra)

import * as fs from 'fs-extra';
import * as path from 'path';

// Write JSON file with pretty formatting
await fs.writeJSON(configPath, config, { spaces: 2 });

// Create directory (won't error if exists)
await fs.ensureDir(path.dirname(schemaPath));

// Write text file
await fs.writeFile(schemaPath, schemaContent);

// Check if file exists
if (await fs.pathExists(configPath)) {
  // File exists
}

Why fs-extra?

  • More user-friendly than Nodeâ€™s built-in fs
  • Promises/async-await (no callbacks!)
  • Extra utilities like ensureDir, pathExists

4. Generating Secure Secrets

import * as crypto from 'crypto';

private generateSecret(): string {
  return crypto.randomBytes(32).toString('hex');
}

What this does:

  • randomBytes(32) - Generates 32 random bytes
  • toString('hex') - Converts to hexadecimal string
  • Result: 64-character secure random string for JWT

Complete Implementation

// packages/cli/src/commands/init.ts
import { Command, Flags } from '@oclif/core';
import inquirer from 'inquirer';
import * as fs from 'fs-extra';
import * as path from 'path';
import * as crypto from 'crypto';
import chalk from 'chalk';

export default class Init extends Command {
  static description = 'Initialize a new DataBridge project';

  static flags = {
    'skip-prompts': Flags.boolean({
      description: 'Skip interactive prompts and use defaults',
      default: false,
    }),
  };

  async run(): Promise<void> {
    const { flags } = await this.parse(Init);

    this.log(chalk.bold.blue('\nðŸš€ DataBridge Project Initialization\n'));

    // Ask user questions or use defaults
    const answers = await this.promptUser(flags['skip-prompts']);

    // Create config file
    await this.createConfig(answers);

    // Initialize Prisma schema
    await this.initPrismaSchema(answers.databaseUrl);

    this.log(chalk.green('\nâœ… DataBridge initialized successfully!\n'));
  }

  private async promptUser(skipPrompts: boolean) {
    if (skipPrompts) {
      return {
        databaseUrl: 'mysql://root:password@localhost:3306/mydb',
        apiOutputPath: 'apps/api/src',
        angularOutputPath: 'src/app/databridge',
        jwtSecret: this.generateSecret(),
      };
    }

    return inquirer.prompt([
      {
        type: 'input',
        name: 'databaseUrl',
        message: 'MySQL database URL:',
        default: 'mysql://root:password@localhost:3306/mydb',
      },
      {
        type: 'input',
        name: 'apiOutputPath',
        message: 'API output path:',
        default: 'apps/api/src',
      },
      {
        type: 'input',
        name: 'angularOutputPath',
        message: 'Angular SDK output path:',
        default: 'src/app/databridge',
      },
    ]).then((answers) => ({
      ...answers,
      jwtSecret: this.generateSecret(),
    }));
  }

  private async createConfig(answers: any) {
    const configPath = path.join(process.cwd(), '.databridge.json');

    const config = {
      version: '0.1.0',
      database: { url: answers.databaseUrl },
      api: {
        outputPath: answers.apiOutputPath,
        port: 3000,
        cors: '*',
      },
      angular: { outputPath: answers.angularOutputPath },
      auth: { jwtSecret: answers.jwtSecret },
    };

    await fs.writeJSON(configPath, config, { spaces: 2 });
    this.log(chalk.green(`âœ“ Created ${configPath}`));
  }

  private async initPrismaSchema(databaseUrl: string) {
    const schemaPath = path.join(process.cwd(), 'prisma', 'schema.prisma');

    if (await fs.pathExists(schemaPath)) {
      this.log(chalk.yellow('âœ“ Prisma schema already exists'));
      return;
    }

    const schemaContent = `datasource db {
  provider = "mysql"
  url      = env("DATABASE_URL")
}

generator client {
  provider = "prisma-client-js"
}

// Your models will appear after running "databridge introspect"
`;

    await fs.ensureDir(path.dirname(schemaPath));
    await fs.writeFile(schemaPath, schemaContent.trim());
    this.log(chalk.green(`âœ“ Created ${schemaPath}`));

    // Create .env file
    const envPath = path.join(process.cwd(), '.env');
    if (!(await fs.pathExists(envPath))) {
      await fs.writeFile(envPath, `DATABASE_URL="${databaseUrl}"\n`);
      this.log(chalk.green(`âœ“ Created ${envPath}`));
    }
  }

  private generateSecret(): string {
    return crypto.randomBytes(32).toString('hex');
  }
}

Command 2: Introspect

Purpose

Connect to your MySQL database and generate Prisma schema automatically by reading:

  • All tables
  • All columns and their types
  • Primary keys, foreign keys
  • Indexes and constraints

Key Concepts

1. Child Process Execution

import { exec } from 'child_process';
import { promisify } from 'util';

const execAsync = promisify(exec);

// Execute shell command and wait for result
const { stdout, stderr } = await execAsync('npx prisma db pull', {
  env: { ...process.env, DATABASE_URL: databaseUrl },
});

What is promisify?

  • Converts callback-based functions to async/await
  • Old way: exec('command', (error, stdout) => {...})
  • New way: await execAsync('command')

2. Environment Variables

const env = {
  ...process.env,           // Keep existing env vars
  DATABASE_URL: databaseUrl, // Add/override DATABASE_URL
};

await execAsync('npx prisma db pull', { env });

Why pass env?

  • Prisma reads DATABASE_URL from environment
  • We can override it without modifying .env file
  • Useful for --db flag

3. Regex Pattern Matching

const modelRegex = /model\s+(\w+)\s*{/g;
const models: string[] = [];
let match;

while ((match = modelRegex.exec(content)) !== null) {
  models.push(match[1]); // match[1] is the captured group (model name)
}

Regex breakdown:

  • model - Literal text â€œmodelâ€
  • \s+ - One or more whitespace characters
  • (\w+) - Capture group: one or more word characters (the model name)
  • \s*{ - Optional whitespace followed by opening brace

Example: model Product { â†’ captures â€œProductâ€

Complete Implementation

// packages/cli/src/commands/introspect.ts
import { Command, Flags } from '@oclif/core';
import { exec } from 'child_process';
import { promisify } from 'util';
import * as fs from 'fs-extra';
import * as path from 'path';
import chalk from 'chalk';

const execAsync = promisify(exec);

export default class Introspect extends Command {
  static description = 'Introspect database schema using Prisma';

  static flags = {
    db: Flags.string({
      description: 'Database URL (overrides config)',
      required: false,
    }),
  };

  async run(): Promise<void> {
    const { flags } = await this.parse(Introspect);

    this.log(chalk.bold.blue('\nðŸ” Introspecting database schema...\n'));

    const config = await this.loadConfig();
    const databaseUrl = flags.db || config.database.url;

    await this.ensurePrismaInstalled();
    await this.runPrismaIntrospection(databaseUrl);

    const models = await this.parseSchema();

    this.log(chalk.green('\nâœ… Schema introspection complete!'));
    this.log(chalk.bold(`ðŸ“Š Found ${models.length} models: `) + 
             chalk.cyan(models.join(', ')));
  }

  private async loadConfig() {
    const configPath = path.join(process.cwd(), '.databridge.json');
    
    if (!(await fs.pathExists(configPath))) {
      this.error('No .databridge.json found. Run "databridge init" first.');
    }

    return fs.readJSON(configPath);
  }

  private async ensurePrismaInstalled() {
    try {
      await execAsync('npx prisma --version');
    } catch (error) {
      this.log(chalk.yellow('âš ï¸  Prisma not found. Installing...'));
      await execAsync('npm install -D prisma @prisma/client');
      this.log(chalk.green('âœ“ Prisma installed\n'));
    }
  }

  private async runPrismaIntrospection(databaseUrl: string) {
    try {
      const env = {
        ...process.env,
        DATABASE_URL: databaseUrl,
      };

      this.log('Running: ' + chalk.cyan('prisma db pull\n'));

      const { stdout, stderr } = await execAsync('npx prisma db pull', {
        env,
        cwd: process.cwd(),
      });

      if (stderr) this.log(chalk.yellow(stderr));
      if (stdout) this.log(stdout);
    } catch (error: any) {
      this.error(
        chalk.red('\nâŒ Failed to introspect database:\n') +
        chalk.gray(error.message) +
        chalk.yellow('\n\nTips:') +
        '\n  â€¢ Check database URL in .databridge.json' +
        '\n  â€¢ Make sure MySQL is running' +
        '\n  â€¢ Verify credentials\n'
      );
    }
  }

  private async parseSchema(): Promise<string[]> {
    const schemaPath = path.join(process.cwd(), 'prisma', 'schema.prisma');

    if (!(await fs.pathExists(schemaPath))) {
      return [];
    }

    const content = await fs.readFile(schemaPath, 'utf-8');
    const modelRegex = /model\s+(\w+)\s*{/g;
    const models: string[] = [];
    let match;

    while ((match = modelRegex.exec(content)) !== null) {
      models.push(match[1]);
    }

    return models;
  }
}
